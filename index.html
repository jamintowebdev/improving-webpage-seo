Meta description

The next meta tag is the description.
It acts like a synopsis on the back of a book. The title is the heading, and the
meta description is an extra couple sentences of information that further
describe what the content is about.
The description is created using a <meta> tag:

<meta name="description" content="Join Now to learn the basics or advance your existing skills.">
